# @magusn/react

> common react components, hooks, etc.

## Install

```bash
yarn add @magusn/react
npm install --save @magusn/react
```

## Usage

```jsx
import React, { Component } from 'react';

import { Button } from '@magusn/react';

function Example(props) {
  return <Button />;
}
```
